# Frontend helper

Cette arborescence est un simple point d'entrée pour lancer le projet Vite existant à la racine.
Les scripts `npm run dev`, `npm run build` et `npm run preview` délèguent aux scripts identiques du dossier parent.

```bash
cd frontend
npm install
npm run dev
```

Les dépendances restent gérées par le projet racine pour éviter les doublons.
